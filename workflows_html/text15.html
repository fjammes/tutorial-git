Création d'une feature branch

  • Création de la feature « helloworld »

$hg flow feature start helloworld

  • Editez un programme hello.py

#!/usr/bin/python

print("Hello World")

  • Rendez le exécutable et exécutez-le

$chmod +x hello.py

$./hello.py

Hello World

  • Ajoutez le fichier au repository

$hg add hello.py

  • Et un petit commit

$hg commit
